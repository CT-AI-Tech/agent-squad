### Skills to Know About
- `/plugin` — Install plugins from the Claude Code marketplace. Used to add tools like arckit, ai-dlc-board-manager, and agent-squad itself.
- `/reload-plugins` — Apply plugin changes in the current session after installing or updating.
- `arckit:start` — Guided onboarding for the arckit enterprise architecture toolkit. Run this first after installing arckit.
- `arckit:init` — Initialize an ArcKit project structure for architecture governance work.
- `agent-squad:implement` — Start a coding loop as the Implementer persona: reads a brief, writes code in your lane, runs tests.
- `agent-squad:finish-feature` — Close out an implementation: validates self-review, checks lane discipline, opens a PR.
- `ai-dlc-board-manager:start-feature` — Pick up a feature from the board and create a branch.
- `ai-dlc-board-manager:sprint-plan` — Propose a sprint from the backlog based on dependencies and priorities.
